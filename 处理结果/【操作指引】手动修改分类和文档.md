# 处理结果文件夹 手动修改操作指引

> 本文件提供手动管理分类和文档的操作指南。

---

## 📁 文件夹结构说明

```
处理结果/
├── V2备份/          ← V2文档原始备份（不可修改）
├── V2归档/          ← V2文档归档存档
├── 历史报告/        ← 历史处理记录
├── 历史文化/        ← 历史、文化相关主题
├── 地理知识/        ← 地理、地图相关主题
├── 摄影技巧/        ← 摄影技术相关
├── 教育育儿/        ← 教育、育儿相关
├── 旅游攻略/        ← 旅游、美食相关
├── 植物养护/        ← 植物、花卉相关
├── 营养健康/        ← 营养、健康相关
└── 其他/            ← 未分类或混合内容
```

---

## ✏️ 分类修改

### 新建分类文件夹
1. 在「处理结果」根目录下新建文件夹
2. 命名规则：`{主题名称}`（简洁明了）
3. 将相关文档移动到新文件夹

### 重命名分类
1. 直接在文件管理器中重命名文件夹
2. 注意：不影响内部文档内容

### 删除分类
1. 确保文件夹内没有重要内容
2. 删除前建议将文档移到其他分类
3. 删除后内容无法恢复，请谨慎

---

## 📄 文档修改

### 移动文档
1. 直接拖拽到目标分类文件夹
2. 或右键 → 剪切 → 粘贴到目标位置

### 重命名文档
1. 命名格式建议：`{分类}-{主题}.md`
2. 例如：`旅游攻略-北京驻京办美食.md`

### 编辑文档内容
1. 用 Markdown 编辑器打开
2. 推荐工具：Typora、VS Code、Notepad++
3. 保存后格式保持 `.md`

### 删除文档
1. 删除后可在「V2备份」中找到原始备份
2. 建议不要直接删除，而是移到「其他」分类

---

## 🔄 恢复操作

### 从备份恢复文档
如果某个分类的文档丢失或损坏：
1. 打开「V2备份」文件夹
2. 找到对应的原始文档
3. 复制到目标分类文件夹

### 查看历史报告
所有处理记录都在「历史报告」文件夹，包括：
- 处理时间
- 处理数量
- 分类结果

---

## 💡 最佳实践

| 操作 | 建议 |
|------|------|
| 新分类命名 | 使用简洁的2-4个字主题名 |
| 文档移动 | 先复制确认，再删除原文件 |
| 批量操作 | 使用 Ctrl 多选后统一移动 |
| 定期整理 | 每月检查一次分类合理性 |

---

## ⚠️ 注意事项

1. **V2备份和V2归档是原始备份**，不建议修改
2. 文档移动后，V7系统下次处理会检测到新位置
3. 如需完全重置，可删除所有分类文件夹，保留备份后重新运行处理
