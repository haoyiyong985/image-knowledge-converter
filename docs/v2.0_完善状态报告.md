## ⚠️ 需要完善的部分

### 高优先级

#### 1. 分类引擎依赖问题
**问题**: 原 `classifier_engine.py` 依赖 sklearn/numpy  
**影响**: 无法直接导入使用

**解决方案**:
```python
# 已实施：自动降级到简化版
try:
    from classifier_engine import ClassifierEngine
except ImportError:
    from classifier_engine_simple import SimpleClassifierEngine as ClassifierEngine
```

**建议**: 
- 短期：使用简化版分类器（已可用）
- 长期：安装 sklearn 获得完整功能

#### 2. 内存监控模块
**问题**: 需要 `psutil` 模块  
**影响**: 内存监控功能受限

**安装命令**:
```bash
pip install psutil -i https://pypi.tuna.tsinghua.edu.cn/simple
```

#### 3. 中文语言包
**问题**: Tesseract 默认只有英文支持  
**影响**: 中文识别效果可能不佳

**解决方案**:
1. 下载中文语言包: https://github.com/tesseract-ocr/tessdata
2. 将 `chi_sim.traineddata` 放入 `C:\Program Files\Tesseract-OCR\tessdata\`
