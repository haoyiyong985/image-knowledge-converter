# OCR引擎自动回退机制说明

**更新日期**: 2026年4月5日  
**版本**: v1.2

---

## 🔄 自动回退流程

当调用OCR引擎时，系统会按以下优先级自动选择和回退：

```
第1优先级: 腾讯云OCR (1000次/月)
    ↓ 额度用完或调用失败
第2优先级: 百度云OCR (1000次/月)
    ↓ 额度用完或调用失败
第3优先级: AI视觉识别 (无限制)
```

---

## 📊 引擎配置

| 引擎 | 优先级 | 免费额度 | 特点 |
|------|--------|----------|------|
| 腾讯云OCR | 1 | 1000次/月 | 精度最高 |
| 百度云OCR | 2 | 1000次/月 | 性价比高 |
| AI视觉识别 | 3 | 无限 | 兜底方案 |

**合计免费额度**: 2000次/月

---

## ⚙️ 回退触发条件

### 1. 额度用完自动回退

当某个引擎的日使用量达到限额时，自动切换到下一个引擎：

```python
# 伪代码
if 腾讯云使用量 >= 1000:
    使用百度云OCR
elif 百度云使用量 >= 1000:
    使用AI视觉识别
```

### 2. 调用失败自动回退

当API调用失败时（如网络错误、权限不足等），自动标记该引擎为"已用完"并切换到下一个：

```python
try:
    result = 腾讯云OCR识别()
except Exception as e:
    标记腾讯云为失败状态
    return 递归调用下一个引擎()
```

### 3. 未配置自动跳过

如果某个引擎没有配置API密钥，自动跳过：

```python
if 未配置腾讯云密钥:
    跳过腾讯云，尝试百度云
if 未配置百度云密钥:
    跳过百度云，直接使用AI视觉
```

## 📝 使用示例

### 基本使用

```python
from ocr_engine_manager import get_ocr_manager

manager = get_ocr_manager()

# 自动选择可用引擎
result, engine_name = manager.recognize_image("图片路径.jpg")

if result == "__AI_VISION__":
    print("云OCR额度已用完，需要使用AI视觉识别")
else:
    print(f"使用 {engine_name} 识别成功")
```

### 查看当前状态

```python
status = manager.get_status()
print(f"当前引擎: {status['current_engine']}")

for engine_id, info in status['engines'].items():
    print(f"{info['name']}: {info['used_today']}/{info['daily_limit']} (剩余: {info['remaining']})")
```

## 💡 使用建议

### 场景1：月处理量 < 1000张
- 只用腾讯云OCR即可
- 精度最高

### 场景2：月处理量 1000-2000张
- 腾讯云 + 百度云组合
- 先用完腾讯云，自动切换到百度云

### 场景3：月处理量 > 2000张
- 前2000张使用云OCR（免费）
- 超出部分使用AI视觉识别
- 或购买百度云次数包（50元/1万次）

## 🔍 故障排查

### 问题：一直使用AI视觉，不调用云OCR

**可能原因**:
1. 云OCR未配置密钥
2. 云OCR额度已用完
3. 云OCR调用失败（权限不足、网络问题）

**检查方法**:
```bash
python ocr_engine_manager.py
```

### 问题：百度云返回"权限不足"错误

**原因**: 百度云应用未开通"通用文字识别"服务

**解决**:
1. 登录 https://console.bce.baidu.com/ai/
2. 进入「文字识别」服务
3. 开通「通用文字识别」
