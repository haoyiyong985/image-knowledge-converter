# 图片知识库自动处理工具 v2.0

## 🎯 功能概述

这是一个**全自动**的图片知识库整理工具，可以：

1. ✅ **自动扫描**待处理图片
2. ✅ **自动识别**图片中的文字（使用腾讯云 OCR）
3. ✅ **自动分类**内容到对应知识库
4. ✅ **自动合并**到现有文档
5. ✅ **自动归档**已处理图片

**无需手动调用 Kimi，一键完成所有操作！**

## 🔧 配置方法

### 方式一：环境变量（推荐）

在 Windows PowerShell 中执行：

```powershell
# 设置环境变量（将 xxx 替换为你的实际密钥）
[Environment]::SetEnvironmentVariable("TENCENT_SECRET_ID", "你的SecretId", "User")
[Environment]::SetEnvironmentVariable("TENCENT_SECRET_KEY", "你的SecretKey", "User")

# 验证设置
$env:TENCENT_SECRET_ID
$env:TENCENT_SECRET_KEY
```

**注意**：设置后需要**重启终端**才能生效。

### 方式二：临时设置

在每次运行前设置：

```powershell
$env:TENCENT_SECRET_ID="你的SecretId"
$env:TENCENT_SECRET_KEY="你的SecretKey"
python auto_process_ocr.py
```

## ⚠️ 常见问题

### Q1: 提示 "未找到腾讯云凭证"

**A**: 请确保已正确设置环境变量 `TENCENT_SECRET_ID` 和 `TENCENT_SECRET_KEY`，并重启终端。

### Q2: OCR 识别不准确

**A**: 
- 确保图片清晰，文字不模糊
- 避免图片过小（建议宽度 > 800px）
- 腾讯云 OCR 对印刷体效果最好

### Q3: 免费额度用完了怎么办

**A**:
- 每月 1000 次免费额度通常足够
- 如需更多，可购买付费套餐（价格很低）
- 或改用其他 OCR 服务（百度、阿里等）

### Q4: 可以处理手写文字吗

**A**: 腾讯云通用印刷体识别主要针对印刷文字，手写识别效果可能不佳。如需手写识别，可使用「通用文字识别（高精度版）」。
