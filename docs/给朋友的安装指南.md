# 给朋友的安装指南

## 📦 如何安装图片知识库转化工具 v1.0

---

## 🎉 简单三步，开始使用

---

### 第1步：下载文件

从我这里获得文件：`image-knowledge-converter_20260318_150338.zip` (39.4 KB)

---

### 第2步：解压到正确位置

#### Windows 用户：

1. **解压 ZIP 文件**
   - 右键点击 ZIP 文件
   - 选择 "解压到..."
   - 得到 `image-knowledge-converter` 文件夹

2. **复制到 Skills 目录**

   打开文件资源管理器，在地址栏输入：
   ```
   %USERPROFILE%\.workbuddy\skills\
   ```

   或者直接输入：
   ```
   C:\Users\你的用户名\.workbuddy\skills\
   ```
   （把"你的用户名"替换成你的实际用户名）

3. **粘贴文件夹**
   - 把解压后的 `image-knowledge-converter` 文件夹
   - 粘贴到上面的 `skills` 文件夹中

---

### 第3步：重启 WorkBuddy

1. 完全关闭 WorkBuddy
2. 重新打开 WorkBuddy
3. 完成！Skill 会自动被识别

## 📁 文件结构说明

安装后，你的电脑上会有这些文件：

### Skills 文件（已安装）
```
C:\Users\你的用户名\.workbuddy\skills\
└── image-knowledge-converter\
    └── SKILL.md          ← WorkBuddy 会自动读取这个文件
```

### 项目文件（需要你创建）
```
D:\图片知识库\
├── 待处理图片\          ← 放新截图的文件夹
│   ├── 主题1\
│   ├── 主题2\
│   └── ...
├── 处理结果\            ← 生成的文档在这里
│   ├── 主题1_文档.docx
│   ├── 主题1_文档.md
│   └── ...
├── 已处理图片\          ← 归档的图片
├── ima_config.txt       ← ima 凭证配置（可选）
└── auto_process.py      ← 核心处理脚本（自动生成）
```

---

## 🚀 开始使用

### 准备工作

1. 创建项目文件夹 `D:\图片知识库\`
2. 在里面创建子文件夹：
   - `待处理图片\` - 放新截图
   - `处理结果\` - 自动生成
   - `已处理图片\` - 自动归档

### 处理截图

1. 把手机截图放到 `待处理图片\主题名\` 下
2. 在 WorkBuddy 中输入：`处理新图片`
3. 等待处理完成
4. 在 `处理结果\` 查看生成的文档

---

## 💡 小技巧

- 批量处理：一次可以放几十张截图，工具会自动识别并分类
- 主题分类：不同主题放在不同文件夹，比如 `健康\`、`工作\`、`学习\`
- 自动归档：处理完的截图会自动移到 `已处理图片\`

## 🤔 遇到问题？

### 问题1：找不到 `.workbuddy` 文件夹
**原因**：你还没打开过 WorkBuddy  
**解决**：先打开一次 WorkBuddy，这个文件夹会自动创建

### 问题2：Skill 没有被识别
**解决**：
1. 检查 `image-knowledge-converter` 文件夹里是否包含 `SKILL.md`
2. 完全关闭 WorkBuddy 再重新打开
3. 点击右上角头像 → Skills 管理 → 刷新

### 问题3：处理图片报错
**解决**：
1. 确保图片是 `.jpg`、`.jpeg` 或 `.png` 格式
2. 确保图片不是损坏的文件
3. 截图错误信息发给我
