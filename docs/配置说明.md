# 配置说明

本文档说明如何配置图片知识库整理工具的 API 密钥。

## 快速开始

### 1. 安装依赖

```bash
python install_deps.py
```

或手动安装：
```bash
pip install python-dotenv requests Pillow pytesseract opencv-python numpy PyYAML python-docx markdown
```

### 2. 配置 API 密钥

**方式一：使用配置向导（推荐）**
```bash
python setup_config.py
```

按提示输入你的 API 密钥即可。

**方式二：手动编辑 .env 文件**

复制 `.env` 文件并填入你的密钥：

```bash
# 腾讯云 OCR
TENCENT_SECRET_ID=你的SecretId
TENCENT_SECRET_KEY=你的SecretKey

# 百度云 OCR
BAIDU_APP_ID=你的AppId
BAIDU_API_KEY=你的APIKey
BAIDU_SECRET_KEY=你的SecretKey
```

### 3. 验证配置

```bash
# 测试百度 OCR
python check_baidu_ocr.py

# 测试所有 OCR 引擎
python config_ocr_env.py
```

## 获取 API 密钥

### 腾讯云 OCR

1. 访问 https://console.cloud.tencent.com/cam/capi
2. 登录腾讯云账号
3. 点击「新建密钥」
4. 复制 SecretId 和 SecretKey

**免费额度**：1000次/月

### 百度云 OCR

1. 访问 https://console.bce.baidu.com/ai/
2. 登录百度账号
3. 进入「文字识别」服务
4. 创建应用，获取 AppID、API Key、Secret Key

**免费额度**：50000次/月

## 文件说明

| 文件 | 说明 |
|------|------|
| `.env` | 配置文件（包含敏感信息，**不要提交到 Git**） |
| `.gitignore` | Git 忽略文件配置 |
| `setup_config.py` | 交互式配置向导 |
| `install_deps.py` | 依赖安装脚本 |
| `check_baidu_ocr.py` | 百度 OCR 配置检查工具 |
| `config_ocr_env.py` | 环境变量配置脚本 |

## 安全提示

1. ✅ `.env` 文件已添加到 `.gitignore`，不会被提交
2. ✅ 所有密钥都通过环境变量读取，不在代码中硬编码
3. ⚠️ 不要分享你的 `.env` 文件给他人
4. ⚠️ 定期更换 API 密钥（建议每 90 天）

## 故障排除

### 问题："未找到配置文件"

**解决**：运行 `python setup_config.py` 创建配置文件

### 问题："API Key 未配置"

**解决**：检查 `.env` 文件是否存在，且包含正确的密钥

### 问题："导入错误: dotenv"

**解决**：运行 `pip install python-dotenv`

### 问题："OCR API 测试失败"

**解决**：
1. 检查网络连接
2. 确认 API 密钥正确
3. 检查是否超出免费额度
