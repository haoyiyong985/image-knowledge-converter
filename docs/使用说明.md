# 图片知识库 — 使用说明

> 适用范围：将手机截图自动整理为 Word + Markdown 知识文档，并自动同步到 ima 个人笔记
> 最后更新：2026年3月16日

---

## 一、目录结构说明

```
D:\新建文件夹\
│
├── 待处理图片\                    ← 新图片放这里（按主题分子文件夹）
│   ├── 健康养生\                  ← 示例：健康类图片放这里
│   └── 旅行记录\                  ← 示例：旅行类图片放这里
│
├── 已处理图片\                    ← 处理完的图片自动归档（不用手动操作）
│
├── 处理结果\                      ← 最终生成的 Word 和 MD 文档
│   ├── 01_抗炎饮食与营养科普.docx/.md
│   ├── 02_肠道健康与饮食分类.docx/.md
│   ├── 03_中医养生与食疗.docx/.md
│   ├── 04_日常饮食建议.docx/.md
│   └── （新文档自动编号添加）
│
├── 【启动提示词】新对话粘贴这段.txt  ← 重启电脑后打开新对话用这个
├── auto_process.py               ← 处理前自动检查脚本（含 ima 同步）
├── ima_sync.py                   ← ima 自动同步脚本
└── ima_config.txt                ← ima API 凭证配置（填入后自动生效）
```

---

## 二、日常使用流程（3步）

### 第 1 步：放入新图片

把手机截图复制到对应主题文件夹：

- 健康养生类 → `待处理图片\健康养生\`
- 旅行记录类 → `待处理图片\旅行记录\`（文件夹没有就新建一个）
- **文件夹名称可以随意起，中英文都行**

---

### 第 2 步：打开 WorkBuddy，恢复背景

> ⚠️ **重要：** WorkBuddy 每次新开对话都是"空白"状态，不记得之前的内容。
> 必须先粘贴启动提示词，AI 才知道你的项目在哪、有哪些文档。

**操作步骤：**

1. 打开电脑上的 WorkBuddy
2. 新建一个对话（或直接在对话框里）
3. 打开文件 `D:\新建文件夹\【启动提示词】新对话粘贴这段.txt`
4. 全选文件内容（Ctrl+A），复制（Ctrl+C）
5. 粘贴到 WorkBuddy 对话框，发送
6. AI 回复"已准备好"后，继续下一步

---

### 第 3 步：发送指令，开始处理

AI 准备好后，发送：

```
处理新图片
```

AI 会自动完成：
1. 读取已有4个文档的章节结构（知道哪些内容已经有了）
2. 逐张识别图片中的文字
3. 判断内容属于哪个分类
4. 同类内容 → 追加到已有文档（不会重复已有章节）
5. 新主题内容 → 自动新建 `05_xxx.docx/.md`
6. 处理完的图片移动到 `已处理图片\` 归档

---

## 三、有新类别文档时怎么操作

当图片内容是全新主题（比如旅行、理财、学习笔记等），AI 会：

1. **自动判断新主题名称**（根据图片内容决定）
2. **自动编号**：在已有04号文档后，新建 `05_旅行记录.md` 和 `05_旅行记录.docx`
3. **在对话中告诉你**：新建了什么文档、包含哪些内容
4. **你无需任何操作**，直接去 `处理结果\` 查看新文件即可

## 四、已有文档的合并规则

| 新内容属于 | 处理方式 |
|-----------|---------|
| 抗炎/坚果/营养素 | 追加到 `01_抗炎饮食与营养科普` |
| 肠道/益生菌/绿灯食物 | 追加到 `02_肠道健康与饮食分类` |
| 中医/养生/食疗/茶饮 | 追加到 `03_中医养生与食疗` |
| 早餐/食谱/饮食建议 | 追加到 `04_日常饮食建议` |
| 全新主题 | 新建 `05_xxx` 文档（自动编号） |

## 五、常用指令

在 WorkBuddy 对话中可以使用：

| 发送内容 | 效果 |
|---------|------|
| `处理新图片` | 处理所有待处理图片 |
| `处理待处理图片/健康养生` | 只处理健康养生文件夹的图片 |
| `查看现有文档` | 列出所有文档和章节列表 |

## 六、注意事项

| 情况 | 处理方式 |
|------|---------|
| 图片文字模糊看不清 | AI 会跳过并告知哪张识别不了 |
| 同一张图跨多个主题 | AI 会拆分内容分别归入各文档 |
| 想重新处理某张图片 | 把图片复制回 `待处理图片\<主题>\`，重新发指令 |
| 电脑重启后找不到对话 | 重新粘贴启动提示词（见第二步）即可 |

## 七、ima 个人笔记自动同步

处理完图片后，文档会**自动同步**到你的 ima 个人笔记，无需任何手动操作。

### 首次配置（只需做一次）

1. 打开文件 `D:\新建文件夹\ima_config.txt`，用记事本打开
2. 把下面两行的占位文字替换成真实值：
   ```
   IMA_CLIENT_ID=填入你的ClientID     ← 改成真实的 ClientID
   IMA_API_KEY=填入你的APIKey         ← 改成真实的 APIKey
   ```
3. 保存文件（Ctrl+S）

> 凭证获取地址：https://ima.qq.com/agent-interface
> 凭证有效期至 2026-04-16，到期后重新获取并更新此文件即可

### 同步逻辑

| 情况 | 处理方式 |
|------|---------|
| 文档首次同步 | 在 ima 新建笔记 |
| 文档有新内容（追加了章节） | 自动在 ima 原笔记末尾追加 |
| 文档未变更 | 跳过，不重复同步 |

### 单独运行同步（可选）

也可以随时手动触发同步，在命令窗口运行：

```
python D:\新建文件夹\ima_sync.py          ← 同步有变更的文件
python D:\新建文件夹\ima_sync.py --force  ← 强制重新同步全部文件
python D:\新建文件夹\ima_sync.py --check  ← 仅验证凭证是否有效
```
