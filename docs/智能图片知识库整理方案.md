# 智能图片知识库整理方案

## 📋 任务概述
将67张图片中的文字内容提取、分类，并整理成Word文档，最终导入ima知识库。

## 🎯 图片内容分析

基于抽样查看，图片主要分为以下几类：

### 1. 健康养生类（占比约80%）
- **食疗养生** - 绿灯食物、健康早餐、营养搭配
- **中医养生** - 三伏天养生、节气养生、穴位保健
- **茶饮养生** - 24节气喝茶指南、药茶配方
- **食疗配方** - 仲冬补心肾、各种养生汤方

### 2. 学习成长类（占比约15%）
- 微信读书截图
- 健康知识科普

### 3. 其他（占比约5%）
- 产品推荐
- 日常记录

## 🔄 优化处理流程

### 方案A：分批处理（推荐）
由于图片数量较多，建议分批处理：

```
批次1: 图片1-10  → 健康养生-食疗篇
批次2: 图片11-20 → 健康养生-中医篇
批次3: 图片21-30 → 健康养生-茶饮篇
批次4: 图片31-40 → 健康养生-配方篇
批次5: 图片41-50 → 健康养生-综合篇
批次6: 图片51-67 → 其他分类
```

### 方案B：按主题自动聚类
根据图片文件名和内容特征自动分组：

1. **微信读书截图** (weread) - 学习成长类
2. **小红书截图** (xhs/xingin) - 健康养生类
3. **微信截图** (wechat/mm) - 健康养生类
4. **相机照片** (Camera/IMG) - 需要人工判断

## 🛠️ 执行步骤

### 第一步：图片预处理
1. 按来源分类图片（微信读书/小红书/微信/相机）
2. 为每批图片生成处理清单

### 第二步：内容提取
使用Kimi-K2.5批量识别图片文字：

**提示词模板：**
```
请帮我识别以下图片中的文字内容，并按格式输出：

图片文件名: [文件名]

提取的文字内容:
[完整文字]

内容分类: [健康养生-食疗/中医/茶饮/配方 或 学习成长 或 其他]

核心知识点:
- [要点1]
- [要点2]
- [要点3]

实用价值: [这段话对读者有什么实际帮助]
```

### 第三步：内容整理
将提取的内容按主题归类，去除重复信息。

### 第四步：生成Word文档
使用Python脚本生成格式化的Word文档。

### 第五步：导入ima知识库
将Word文档导入到ima知识库中。
