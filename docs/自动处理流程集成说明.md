## 故障排查

### 启动提示词更新失败

**可能原因**:
1. `update_prompt.py` 不存在
2. 处理结果目录中没有编号文档

**解决方法**:
```bash
# 检查脚本是否存在
ls update_prompt.py

# 手动运行测试
python update_prompt.py
```

### OCR识别失败

2. 网络连接问题

**解决方法**:
```bash
# 检查OCR配置

python ocr_recognize.py "图片路径.jpg"
```

## 注意事项

1. **全自动运行**: 处理过程中不会出现"是否继续"的提示，所有批次会依次自动完成
2. **状态持久化**: 分批处理状态保存在 `.workbuddy/memory/batch_state.json`
3. **日志记录**: 处理日志保存在 `.workbuddy/memory/processing_log.json`
4. **错误处理**: 单个批次失败不会影响其他批次的处理
