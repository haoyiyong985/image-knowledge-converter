# 智能分批全自动处理方案 v1.2

**制定时间**: 2026-03-24  
**核心原则**: 固定12张/批次，全自动连续处理，零人工干预

---

## 方案概述

### 分批策略

| 参数 | 设置 |
|------|------|
| 每批数量 | 固定12张 |
| 分批依据 | 不区分图片大小 |
| 处理方式 | AI自动连续处理 |

### 分批示例

| 总图片数 | 批次分配 | 处理流程 |
|---------|---------|---------|
| 8张 | 1批(8张) | 直接处理8张 |
| 20张 | 2批(12+8) | 先处理12张 → 再处理8张 |
| 36张 | 3批(12+12+12) | 连续处理3批 |
| 50张 | 5批(12+12+12+12+2) | 连续处理5批 |

## 脚本实现

### 新增脚本: `auto_batch_processor.py`

```python
# 核心功能
1. scan_images() - 扫描所有待处理图片
2. calculate_batches(images, batch_size=12) - 计算批次
3. process_batch(batch_images) - 处理单个批次
4. process_all_batches() - 连续处理所有批次
5. finish_workflow() - 统一收尾
```

### 处理流程代码

```python
def process_new_images():
    # 1. 扫描
    images = scan_all_images()
    total = len(images)
    print(f"发现 {total} 张待处理图片")
    
    # 2. 计算批次 (固定12张/批)
    batch_size = 12
    batches = [images[i:i+batch_size] for i in range(0, total, batch_size)]
    print(f"分 {len(batches)} 批处理，每批最多12张")
    
    # 3. 连续处理所有批次
    all_results = []
    for i, batch in enumerate(batches, 1):
        print(f"\n========== 第{i}批 ({len(batch)}张) ==========")
        results = process_batch(batch)  # OCR + 分类
        all_results.extend(results)
        print(f"进度: {min(i*batch_size, total)}/{total}")
    
    # 4. 统一收尾
    finish_workflow()
    print("\n✅ 全部处理完成！")
```
