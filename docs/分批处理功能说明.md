# 智能分批处理功能说明

## 功能概述

`batch_processor.py` 是图片知识库工具的智能分批处理模块，提供以下核心功能：

1. **智能分批** - 根据图片大小自动分批
2. **断点续传** - 中断后可从上次位置继续
3. **进度追踪** - 实时显示处理进度
4. **失败重试** - 自动重试失败的批次

---

## 使用方法

### 1. 初始化处理会话

```bash
python batch_processor.py init
```

- 扫描所有待处理图片
- 按文件夹和大小智能分批
- 创建处理状态记录
- 如果有未完成的会话，会自动恢复

### 2. 获取下一个批次

```bash
python batch_processor.py next
```

- 返回下一个待处理的批次信息
- 自动将批次状态改为"处理中"
- 包含批次ID、文件夹、图片列表

### 3. 查看处理进度

```bash
python batch_processor.py progress
```

显示：
- 会话ID
- 总批次数
- 已完成/处理中/待处理/失败数量
- 完成百分比

### 4. 标记批次完成

```bash
python batch_processor.py complete <批次ID>
```

例如：
```bash
python batch_processor.py complete 示范_001
```

### 5. 标记批次失败

```bash
python batch_processor.py fail <批次ID>
```

- 失败批次会自动重试（最多3次）
- 超过3次后状态变为"失败"

### 6. 重置失败批次

```bash
python batch_processor.py reset
```

- 将所有失败批次重置为"待处理"
- 重试计数清零

### 7. 清除处理状态

```bash
python batch_processor.py clear
```

- 删除处理状态文件
- 下次将创建新会话

## 分批策略

| 图片大小 | 批次大小 | 说明 |
|---------|---------|------|
| < 500KB | 10张/批 | 小图片，可以多张一起处理 |
| 500KB - 2MB | 6张/批 | 中等图片 |
| 2MB - 5MB | 4张/批 | 大图片 |
| > 5MB | 2张/批 | 超大图片，减少每批数量 |

## 状态文件

处理状态保存在：
```
D:\新建文件夹\.workbuddy\memory\batch_state.json
```

包含：
- 会话ID和开始时间
- 所有批次信息
- 每张图片的路径和哈希值
- 批次处理状态和时间戳

## 工作流程示例

### 完整处理流程

```bash
# 1. 初始化
python batch_processor.py init

# 2. 循环处理每个批次
while true; do
    # 获取下一批次
    python batch_processor.py next
    
    # 处理图片（这里调用实际的OCR和处理逻辑）
    # ...
    
    # 标记完成
    python batch_processor.py complete 示范_001
done

# 3. 查看最终进度
python batch_processor.py progress
```

### 中断续传流程

```bash
# 假设上次处理到一半中断了
# 直接重新初始化，会自动恢复
python batch_processor.py init

# 显示：发现未完成的会话: 20260323_192826
#       已完成: 1/5

# 继续获取下一批次
python batch_processor.py next
```

## 在Python代码中使用

```python
from batch_processor import BatchProcessor

# 创建处理器
processor = BatchProcessor()

# 初始化
state = processor.initialize()

# 循环处理
while True:
    batch = processor.get_next_batch()
    if not batch:
        break
    
    # 处理批次中的图片
    for img_path in batch.images:
        # 调用OCR识别
        # 整理内容
        pass
    
    # 标记完成
    processor.mark_batch_completed(batch.batch_id)

# 查看进度
processor.print_progress()
```

---

## 注意事项

1. **不要手动删除状态文件** - 除非你想重新开始
2. **同一图片不会被重复处理** - 通过文件哈希检测
3. **支持多文件夹** - 每个文件夹独立分批
4. **自动去重** - 相同图片只处理一次

## 故障排除

### 问题：状态文件损坏

**解决**：
```bash
python batch_processor.py clear
python batch_processor.py init
```

### 问题：批次一直失败

**解决**：
```bash
# 查看失败原因
python batch_processor.py progress

# 重置后重试
python batch_processor.py reset
```

### 问题：想重新开始

**解决**：
```bash
python batch_processor.py clear
python batch_processor.py init
```
