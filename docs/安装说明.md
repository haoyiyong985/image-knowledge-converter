## 🚀 如何使用

### 方法1：自动识别（推荐）

WorkBuddy 会自动识别新安装的 Skill，你可以直接使用以下指令：

```
初始化知识库
```
```
处理新图片
```
```
处理待处理图片/主题名
```
```
查看现有文档
```
```
工具优化
```

### 方法2：手动触发

如果自动识别失败，可以手动加载 Skill：

1. 打开 WorkBuddy
2. 点击右上角头像 → **Skills 管理**
3. 找到 **图片知识库转化工具 (image-knowledge-converter)**
4. 点击 **启用**

**解决**：

2. 或点击右上角头像 → Skills 管理 → 刷新

**解决**：
1. 检查 `D:\新建文件夹\待处理图片\` 路径是否正确

**解决**：
1. 检查 `D:\新建文件夹\处理结果\` 文件夹
2. 查看处理日志（每次处理会生成日志）
