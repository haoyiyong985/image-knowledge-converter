## 后续使用

### 新图片处理
使用V7系统处理新图片时，AI会自动：
1. 分析图片内容
2. 判断分类（可用预设分类或新建）
3. 生成V7格式文档名
4. 归档到对应分类文件夹

### V2内容复用
如果新图片内容属于V2已有的分类：
1. V7会检测相似文档
2. 自动追加内容到对应文档
3. 无需重新创建

### 备份恢复
如需恢复V2原始文档，使用 `V2备份/` 或 `V2归档/` 文件夹。
