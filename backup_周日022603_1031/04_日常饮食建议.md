## 二、全日食谱推荐

### 标准成人一日饮食量

| 食物类别 | 推荐摄入量 | 备注 |
|---------|-----------|------|
| **粮食** | 200g | 其中粗粮50g——绿豆 |
| **鸡蛋** | 1个 | 优质蛋白来源 |
| **牛奶** | 1袋 | 钙质补充 |
| **肉类** | 100g | 动物蛋白 |
| **虾仁** | 25g | 海鲜蛋白 |
| **蔬菜** | 500g | 膳食纤维和维生素 |
| **北豆腐** | 50g | 植物蛋白 |
| **水果** | 200g | 维生素和抗氧化物 |
| **油** | 20g | 健康脂肪 |
| **盐** | 5g | 控制钠摄入 |

### 营养搭配要点

1. **粗细搭配**：主食中粗粮占1/4左右
2. **荤素搭配**：动物蛋白和植物蛋白结合
3. **五色蔬菜**：每天摄入不同颜色的蔬菜
4. **适量水果**：控制糖分摄入
5. **少油少盐**：每日油盐控制在推荐量内
