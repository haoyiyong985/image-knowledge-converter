# 智能内容分类系统

## 设计原则

不再跳过任何内容，所有图片都根据内容自动分类并创建文档。

## 分类体系

### 现有分类（饮食健康类）
1. **01_抗炎饮食与营养科普** - 脂肪酸、维生素、抗氧化、营养素
2. **02_肠道健康与饮食分类** - 益生菌、膳食纤维、消化
3. **03_中医养生与食疗** - 食疗、节气、中药、体质
4. **04_日常饮食建议** - 食谱、搭配、膳食宝塔
5. **05_中医经络与穴位** - 穴位、经络、取穴
6. **06_7日益肝食谱** - 护肝、食谱、一周计划
7. **07_中医养生方子** - 调理方、草药、降脂

### 新增分类（扩展知识库）
8. **08_疾病预防与健康管理** - 疾病预防、体检指标、健康筛查
9. **09_急救与健康安全** - 急救知识、中风识别、安全防护
10. **10_医学常识与检查** - 医学知识、检查指标、诊断标准
11. **11_生活技巧与常识** - 烹饪技巧、生活窍门、实用技能

## 新文档命名规则

当内容不属于现有任何分类时，自动创建新文档：

```
XX_内容主题.md
```

命名原则：
1. 使用简洁中文
2. 体现内容核心主题
3. 避免笼统词汇如"新主题"

## 实施计划

1. 扩展分类体系，增加非饮食内容分类
2. 修改AI处理逻辑，所有内容都分类处理
3. 自动创建新文档时，根据内容智能命名
4. 更新启动提示词，包含所有分类说明
