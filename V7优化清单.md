## ✅ V7 新设计规范

### 分类体系

```
预设分类：历史文化、营养健康、生活方式、教育育儿、旅游攻略、综合知识、其他
动态分类：AI判断不属于预设分类时自动新建
```

### 文档命名规范

```
格式：{分类}-{主题}
示例：历史文化-钱谦益传、营养健康-春季养生
```

### 处理流程

```
图片 → OCR → AI分析
              ↓
        ┌─────────────┐
        │ 提取分类    │ ← 判断属于预设分类还是新建
        │ 提取主题词  │ ← 人名/事件名/核心主题（合并到主主题）
        │ 提取内容摘要│
        └─────────────┘
              ↓
        检查是否存在同名文档
              ↓
        ┌─────────────┴─────────────┐
        ↓                           ↓
     存在同名                    不存在
        ↓                           ↓
   追加内容                    新建文档
```
