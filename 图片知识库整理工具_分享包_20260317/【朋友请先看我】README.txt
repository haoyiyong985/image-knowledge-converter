欢迎使用「图片知识库整理工具」！

这个工具可以帮你把手机截图自动整理成 Word 文档和 Markdown 文件，
并可选择自动同步到 ima 个人笔记。

【第一步】请先打开这个文件：
  图片知识库_配置说明（小白版）.docx

  里面有详细的图文配置步骤，大约 10 分钟可以配置完成。

【包含文件清单】
  1. 图片知识库_配置说明（小白版）.docx  ← 从这里开始
  2. auto_process.py                      ← 核心处理脚本
  3. ima_sync.py                          ← ima 自动同步脚本
  4. ima_config.txt                       ← ima API 凭证配置（填完即用）
  5. test_flow.py                         ← 验证配置是否正确
  6. 使用说明.md                          ← 日常操作手册
  7. 【启动提示词】新对话粘贴这段.txt    ← 每次新对话时用

【需要的软件】
  - Python 3.8+（免费，官网下载）
  - WorkBuddy（AI 对话工具）
  - ima PC 客户端（可选，用于知识库管理）

祝使用愉快！
