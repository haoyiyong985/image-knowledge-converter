## 分类系统自动新增功能 - 已完成

**时间**: 2026年4月6日 09:25

### 背景
用户要求：分类配置系统需要在遇到已有分类之外的内容时，能根据内容自动添加新分类。

### 完成内容

**修改 `image_classifier.py`，新增以下方法：**

1. `_extract_topic_keywords(text, top_n=15)`
   - 从文本中提取2-6字的高频中文词组
   - 过滤150+个通用停用词
   - 按频次排序返回关键词列表

2. `auto_add_category(text, category_name=None, keywords=None, document=None, ...)`
   - 主动触发接口，可手动指定分类名称
   - 自动提取关键词 → 推断分类名 → 生成唯一ID → 追加到 YAML
   - 若同名分类已存在，自动合并关键词（去重）

3. `classify_with_auto_add(text, auto_add_threshold=0.2, min_text_length=30)`
   - 一体化接口：先分类，若置信度低于阈值且文本够长，自动触发新增
   - 新增后重新分类，选用更好的结果

4. `_append_category_to_yaml(new_category)` - 追加写 YAML，保留原有格式
5. `_save_config()` - 整体覆写 YAML（合并关键词场景）
6. `_merge_keywords_to_existing(cat, new_keywords)` - 向已有分类追加新关键词

**新增命令行命令：**
- `python image_classifier.py auto-add --text '...'` - 手动触发新增
- `python image_classifier.py auto-add --text '...' --name '分类名'` - 手动命名
- `python image_classifier.py test-auto` - 测试自动新增功能

### 功能验证结果
测试通过 6/6 (100%)：
- 测试1: 汽车保养（陌生内容）→ 自动新增触发 [PASS]
- 测试2: 宠物护理（手动命名）→ 新增成功 [PASS]
- 测试3: 重新加载验证持久化 → 2 个自动分类保留 [PASS]
- 测试4: 向已有分类追加关键词 → 追加 7 个关键词 [PASS]
- 测试5: 原有分类（肠道健康）不受影响 → 置信度 59.3% [PASS]

## P0-4 智能分类器引擎完成

**时间**: 2026年4月6日 10:09

### 完成内容

**1. 创建 `scripts/classifier_engine.py`：**
- `ClassifierEngine` 类：智能分类器核心引擎
- `ClassificationResult` 数据类：分类结果数据结构
- 支持关键词匹配、语义相似度（可选）、多重分类、置信度评分

**API 接口**：
```python
from scripts.classifier_engine import ClassifierEngine

engine = ClassifierEngine()

# 单分类（返回最高置信度结果）
result = engine.classify("抗炎饮食金字塔", threshold=0.5)

# 多分类（返回多个候选）
results = engine.classify(content, threshold=0.3, multi_class=True, max_results=3)

# 快速建议
suggestions = engine.suggest("中医养生", top_k=3)

# 详细解释
explanation = engine.explain_classification(content)
```

**2. 创建 `scripts/classify.py`：**
命令行分类管理工具

```bash
# 分类
python scripts/classify.py classify "宝宝辅食添加"

# 获取建议
python scripts/classify.py suggest "中医养生" --top-k 5

# 列出所有分类
python scripts/classify.py list

# 分类详情
python scripts/classify.py info nutrition

# 详细解释
python scripts/classify.py explain "抗炎饮食金字塔"

# 批量分类
python scripts/classify.py batch content.txt
```

### 验证结果
- 成功加载 10 个分类
- 测试匹配率 4/5 (80%)
- 多重分类功能正常
- 置信度评分正常
- 命令行工具正常运行

### 功能特性
1. **关键词匹配**: 提取内容关键词，与配置匹配
2. **语义相似度**: 可选启用（需要 sentence-transformers）
3. **多重分类**: 一张图片可属于多个分类
