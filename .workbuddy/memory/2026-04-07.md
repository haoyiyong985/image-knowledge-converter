### 分批配置
| 图片大小 | 每批数量 |
|---------|---------|
| <500KB | 10张/批 |
| 500KB-2MB | 6张/批 |
| 2MB-5MB | 4张/批 |
| >5MB | 2张/批 |

### 使用方式
```bash
python auto_process_all_v6.py           # 全自动处理
python auto_process_all_v6.py --batch    # 分批模式（可中断续传）
python auto_process_all_v6.py --init     # 仅初始化分批
python auto_process_all_v6.py --progress # 查看进度
python auto_process_all_v6.py --clear    # 清除分批状态
```
