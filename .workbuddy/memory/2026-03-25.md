## 归档功能优化 v2.1（19:30）

### 优化内容

根据用户建议，对 `simple_batch_processor.py` 进行以下优化：

#### 1. 增加归档重试机制

```python
# 新增配置参数

```

**重试逻辑**：
- 遇到 `PermissionError`（文件被占用）时自动重试
- 重试延迟递增：0.5s → 1.0s → 1.5s
- 最大重试3次后仍失败则标记为失败

#### 2. 增加失败文件列表

```python
# 返回结果增加失败文件列表
archive_images() -> Tuple[int, int, List[str]]
# (成功数, 失败数, 失败文件列表)
```

#### 3. 增加详细失败提示

归档失败时显示：
- 失败文件完整列表
- 失败原因分析（权限错误/文件占用）
- 具体操作建议（关闭占用程序/手动移动）

#### 4. 新增验证命令 `--verify`

```bash
python scripts/simple_batch_processor.py --verify
```

功能：
- 检查待处理文件夹是否还有未归档文件
- 显示未归档文件列表（最多10个）
- 给出归档建议

#### 5. 自动处理重名文件

归档时如果目标文件已存在，自动添加数字后缀：
- `image.jpg` → `image_1.jpg` → `image_2.jpg`
