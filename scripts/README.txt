# 图片知识库转化工具 1.0

## 简介

这是一个全自动的图片知识库整理工具，可以将手机截图中的文字识别出来，按主题自动分类整理成 Word 文档和 Markdown 文件，并支持自动同步到 ima 个人笔记。

## 功能特点

- **智能分类**：自动判断图片内容属于哪个主题，追加到对应文档
- **自动归档**：处理完成后自动将图片从"待处理"移动到"已处理"
- **双重输出**：同时生成 Word (.docx) 和 Markdown (.md) 格式
- **ima 同步**：可选配置，自动将文档同步到 ima 个人笔记（进阶功能）
- **持续优化**：支持通过"工具优化"指令不断改进工具能力

## 核心文件

| 文件 | 说明 |
|------|------|
| `auto_process.py` | 核心处理脚本，每次处理新图片时运行 |
| `ima_sync.py` | ima 自动同步脚本（可选） |
| `ima_config.txt` | ima API 凭证配置（首次配置时填写） |
| `test_flow.py` | 配置验证脚本 |
| `使用说明.md` | 日常操作手册 |
| `【启动提示词】新对话粘贴这段.txt` | WorkBuddy 启动提示词（每次新对话时用） |
| `README.txt` | 项目说明文档（本文件） |

## 使用流程

### 1. 放入新图片
将手机截图放入 `待处理图片\<主题名>\` 文件夹

### 2. 启动 WorkBuddy
粘贴启动提示词内容，发送"处理新图片"

### 3. 自动处理
工具自动识别、分类、整理、归档，完成后自动同步到 ima（如已配置）

## 常用指令

- `处理新图片` → 全量扫描处理所有新图片
- `处理待处理图片/<主题名>` → 指定主题处理
- `查看现有文档` → 查看文档结构和章节
- `工具优化` → 对现有转化工具进行技能提升
